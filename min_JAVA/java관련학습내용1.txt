python, javascript: 인터프리터 언어(interpreter language) -> 컴파일 X
java, c++: 컴파일 언어(compile language) -> 컴파일 필요
  -> java같은 경우 javac프로그램 통해서 기계어로 바꾸어주고
      바뀌어진 기계어 파일을 java프로그램 통해서 실행

자바 개발 환경 IDE(Integrate Development Enviroment)
  (1) Eclipse(이클립스), STS3~4(Spring Tool Suite) -> 장점: 무료, 어느정도 plugin이 있다.
                                                                단점: 느리다, 버그가 많다
  (2) Intellij(인텔리제이) -> 장점: 버그가 많이 없다. 편리한 도구를 많이 제공
                                           무료버전이 있다.
                                   단점: 유료, 약간 무겁다.
  (3) Visual Studio Code -> 장점: 무료, 어느정도 plugin이 있다., 가볍다
                                   단점: Intellij보다 손이 많이 간다.
